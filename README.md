# Task API

Esta é uma API de gerenciamento de tarefas construída com Node.js, Express e TypeScript.

## Instalação

npm install
npm run dev
http://localhost:3333/

Endpoints

1. Criar Tarefa
   Corpo da Requisição (JSON):
   {
   "title": "Nome da Tarefa"
   }

2. Listar Todas as Tarefas

3. Excluir Tarefa

4. Atualizar Tarefa
